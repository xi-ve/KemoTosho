# ShiruExtension_ToshoClone

Shiru extension for Anime Tosho Clone API.

## Installation

1. Open Shiru and go to **Settings > Extensions > Sources**
2. Add this source: `gh:xi-ve/ShiruExtension_ToshoClone`
3. Ensure your Anime Tosho Clone API is running (default: http://localhost:8002)

## Configuration

Edit `BASE_URL` in `sources/animetosho-clone.js` if your API runs elsewhere.
